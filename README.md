# todo-react-app

## Installation
git clone https://github.com/italo2407/todo-react-app.git

cd todo-react-app

npm i

cd client/

npm i

cd server

npm i

## Usage

cd todo-react-app

npm start

if you only run the client side run the following:

### Client side

npm run client

if you only run the server side run the following:

### Server side

npm run server
